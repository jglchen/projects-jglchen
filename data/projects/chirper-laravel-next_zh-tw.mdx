[The Laravel Bootcamp](https://bootcamp.laravel.com/) 展示了[使用 Blade](https://bootcamp.laravel.com/blade/installation)、[使用 Livewire](https://bootcamp.laravel.com/livewire/installation) 以及[使用 JavaScript 和 Inertia](https://bootcamp.laravel.com/inertia/installation) 建立名為 Chirper 的微型部落格平台的三種不同方法。

本專案嘗試**使用 Laravel Chirper API 作為後端以為設定身份驗證之用，並使用 Next.js 作為瀏覽器前端**。
Laravel Chirper API 包括 Laravel Breeze API，它是個簡單小型的程式集，執行 Laravel 所有的身份驗證功能，包括登入、註冊、密碼重設、電子郵件驗證和密碼確認。Sanctum 是一個簡單的配套，您可以使用它向使用者頒發 API 符記，而無需煩複的認證。當使用 API 符記發出請求時，符記應作為 Bearer 符記被包含在授權(Authorization)標頭中。

本專案基本上遵循Laravel Bootcamp 所展示的構思，並添加了一些額外的功能，如下所示：
- Chirper 顯示網頁載入了捲動分頁機制。
- Chirper 頁面載入後將定期更新內容。
- 提供用戶追蹤和取消追蹤其他用戶的機制。
- Laravel Bootcamp 展示過當用戶建立一則 Chirp 時系統會發送電子郵件通知給其他用戶的機制，在本專案中，我們將電子郵件通知限制為僅向該用戶的追蹤者發送。
