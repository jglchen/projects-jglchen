使用 **React.js** 建立 Markdown 編輯器應用程式是為了方便編輯 Markdown 文件。 **WYSIWYG**編輯器允許對 Markdown 缺乏經驗的使用者使用熟悉的工具列按鈕和快捷方式。 此外，編輯 Markdown 文件時會同時顯示 HTML 輸出。

Markdown 不是一般使用者熟悉的文法，編輯時視覺上也不清晰。 換句話說，對於不熟悉的使用者來說，他們編寫的語法在他們查看結果之前沒有什麼意義。 本應用程式旨在為不太熟悉或剛學習 Markdown 語法的非技術用戶彌補這一差距。

本應用程式採用**Responsive web  CSS 設計**，這使得應用程式在所有裝置上看起來都很好。 我們也同時提供 **iOS** 和 **Android** 行動應用程式。 這些應用程式是使用**React Native** 開發的，任何有興趣的人都可以透過**[Expo Publish Link](https://expo.dev/@jglchen/md-editor)** 使用 [Expo Go](https://expo.dev/client) 測試這些應用程式。
