Laravel Breeze 執行 Laravel 所有的身份驗證功能，包括登入、註冊、密碼重設、電子郵件驗證和密碼確認。Breeze 包括一個簡單的「個人資料」頁面，使用者可以在其中更新其姓名、電子郵件地址和密碼。

本專案嘗試使用 **Next.js 全端應用功能來模擬 Laravel Breeze**。
Laravel Breeze 的預設視圖層包含使用 Tailwind CSS 進行樣式設計的簡單 Blade 樣本。Next.js React 前端網頁採用 Tailwind CSS 來進行樣式設計，並遵循 Laravel Breeze 的預設視圖設計。
在這個專案中，Prisma作為ORM用於管理資料庫，就像Eloquent用來連接Laravel和資料庫。
