Laravel Breeze 執行 Laravel 所有的身份驗證功能，包括登入、註冊、密碼重設、電子郵件驗證和密碼確認。Breeze 包括一個簡單的「個人資料」頁面，使用者可以在其中更新其姓名、電子郵件地址和密碼。

Laravel Breeze 的預設視圖層包含使用 Tailwind CSS 進行樣式設計的簡單 Blade 樣本。或者，Breeze 可以使用 Vue 或 React 配合 Inertia 來架構其應用程式。
Laravel Breeze 也可以建立一個身份驗證 API，針對 Next 等現代 JavaScript 應用程式進行身份驗證。

本專案嘗試**使用 Laravel Breeze API 作為後端以為設定身份驗證之用，並使用 Next.js 作為瀏覽器前端**。Laravel Sanctum 被 Laravel Breeze 引入以提供身份驗證系統。
Sanctum 是一個簡單的配套，您可以使用它向使用者頒發 API 符記，而無需煩複的認證。
當使用 API 符記發出請求時，符記應作為 Bearer 符記被包含在授權(Authorization)標頭中。Next.js React 前端網頁採用 Tailwind CSS 來進行樣式設計，並遵循 Laravel Breeze 的預設視圖設計。
